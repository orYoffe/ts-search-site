# Typescript search website

## Installation

```sh
# inside the root directory
$ npm i
```

## Running

### Development mode

```sh
$ npm run dev
```

### Production build

```sh
$ npm run build && npm run start
```

Then open your browser at [http://localhost:3000](http://localhost:3000)

## Tests

### Watch mode

```sh
$ npm run test <optional path and other jest flags>
```

### CI mode (all tests)

```sh
$ npm run test:ci
```
